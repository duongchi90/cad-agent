# File IPC Phase 4 End-to-End Smoke Test Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Verify build, review, deliberate mismatch, repair, and final review through real AutoCAD File IPC.

**Architecture:** Add one opt-in unittest guarded by `CAD_AGENT_FILE_IPC=1`; it creates a temporary DXF and uses `FileIPCLiveMCPClient` with the AutoCAD frame handle. Default test runs skip it.

**Tech Stack:** Python 3.11, unittest, ezdxf, AutoCAD File IPC.
